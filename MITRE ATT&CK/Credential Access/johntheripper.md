# John The Ripperとは
主にパスワードクラッキングに使用される、無料でオープンソースのソフトウェアツール。

## クラック
```bash
john <ファイル名> -w=<ワードリストのパス> # 基本的な使用
--format=<フォーマット> # フォーマットを指定
 raw-md5 # md5ハッシュ
 md5crypt # $1$から始まるmd5
--fork=4 # 4スレッドで実行
```

## 変換コマンド
```bash
ssh2john <sshキー> > <出力名> # ssh2ファイルをハッシュに変換
zip2john <Zipファイル> > <出力名> # ロックされているzipファイルをハッシュに変換
keepass2john <keepassファイル> > <出力名> # ロックされているkeepassファイルをハッシュに変換
```
