# MySQLとは
世界で最も広く利用されているオープンソースのRDBMSの一つ。

## ポート
3306
33060(最新機能であるXプロトコルとMySQL Shellに対応している場合)

## 接続方法
```bash
mysql -h <ターゲット> -P <ポート> -u <ユーザー名> -p<パスワード> # パスワードと-pの間にスペース不要
mysql -h <ターゲット> -P <ポート> -u <ユーザー名> -p
mysql -h <ターゲット> -P <ポート> -u <ユーザー名> -p <データベース名> # データベースを指定
mysql -u <ユーザー名> -p<パスワード> -e '<SQLクエリ>;' # ローカルのMySQLに接続と同時にSQLを実行
-ssl=0 # 意図的にTLS/SSL接続をしない
```

## コマンド
```bash
show databases;
use <データベース名>;
show tables;
```
